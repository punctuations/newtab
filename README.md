## Start development

#### Extension

```bash
cd extension

yarn
# or
npm i
```

##### development

```bash
yarn start
# or
npm run start
```

##### build

```bash
yarn build
# or
npm run build
```

----

#### TAB

```bash
cd tab

yarn
# or
npm i
```

##### development

```bash
yarn dev
# or
npm run dev
```

##### build

```bash
yarn build
# or
npm run build
```
